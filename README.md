# Personal Portfolio

A modern, minimalist portfolio website showcasing my journey as a Full Stack Developer. Built with clean design principles and smooth animations to create engaging digital experiences.

## About

This portfolio represents my evolution from a coding enthusiast (started in 2021) to a dedicated full-stack developer pursuing Software Engineering at iCET. The site features a clean, professional design with a focus on user experience and modern web technologies.

### Key Features

- **Responsive Hero Section** - Eye-catching introduction with profile image and call-to-action
- **Animated Tech Stack Slider** - Showcasing 18+ technologies with smooth infinite scroll animation
- **About Section** - Personal journey and development focus areas
- **Featured Projects** - Highlighting production-ready applications with detailed tech stacks
- **Social Links Integration** - Direct connections to professional profiles
- **Modern UI/UX** - Clean design with custom animations and grain texture overlay

### Technologies Showcased

**Frontend:**

- HTML5, CSS3, JavaScript
- React, Next.js, Tailwind CSS

**Backend:**

- Node.js, Express.js
- Java, Spring Boot

**Databases:**

- MongoDB, MySQL, PostgreSQL

**Tools & Others:**

- TypeScript, Git, GitHub, Postman, Figma

## Design Highlights

- **Color Palette:** Minimalist design with warm neutral tones (#d7d5d3, #ffffff, #242424)
- **Typography:** Roboto font family for clean readability
- **Animations:** Smooth transitions, infinite tech stack slider, hover effects

- **Visual Effects:** Custom grain texture overlay for added depth

## Featured Projects

### 1. Blog API

Production-style REST API with JWT authentication, refresh tokens, MongoDB integration, Cloudinary uploads, centralized logging, rate limiting, and Nginx + PM2 deployment.

**Tech Stack:** TypeScript, Node.js, Express, MongoDB, JWT, Nginx

### 2. Customer Management System

Full-stack application with Spring Boot backend, JPA/Hibernate ORM, MySQL database, Docker containerization, and environment-based configuration.

**Tech Stack:** Java, Spring Boot, MySQL, JPA, Docker, Hibernate
